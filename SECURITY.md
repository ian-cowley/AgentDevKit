# Security Policy

## Supported Versions

Specifically, for the `AgentDevKit`, only the latest stable version of the library is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it via [GitHub Issues](https://github.com/ian-cowley/AgentDevKit/issues).

### What to Include
When reporting a vulnerability, please provide the following details:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact of the vulnerability.

### Our Commitment
We take security seriously and will investigate and address reported vulnerabilities as quickly as possible. 

---

Thank you for helping keep AgentDevKit secure!
